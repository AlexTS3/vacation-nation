This is personal AngularJS project for project defence at SoftUni

It has dynamic rendering for logged in users and guests

If you are logged in you can:

1 - Add places;

2 - Delete your own places;

3 - Edit your own places;/p>

4 - Add places to your plans;

5 - See your planned places;

6 - Remove planned places;

7 - See the places you have added;

8 - See details for particular place;

If you are guest you can:

1 - Search places;

2 - See all places added by users;

3 - Register as user;

4 - Login as existing user;

5 - See details for particular place;
